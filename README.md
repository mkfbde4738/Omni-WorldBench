## 🛠️ Troubleshooting

### The app does not open

- Check that the file finished downloading
- Try running it again as a normal user
- Right-click the app and choose Run as administrator
- Update your graphics driver
- Restart your PC and try again

### Windows blocks the file

- Open the file properties
- If you see an Unblock box, select it
- Apply the change
- Try opening the file again

### The benchmark runs slowly

- Close other apps
- Free up disk space
- Use a smaller test set
- Update Windows
- Check that your PC meets the memory and CPU needs

### Results do not save

- Make sure the output folder exists
- Check that you have write access to the folder
- Pick a folder on your local drive
- Avoid saving to a full drive or network location
